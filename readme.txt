== Description ==
امکان پرداخت از هرجا و هرزمان : امکان پرداخت آنلاین و شخصی از تمام نقاط کشور
دریافت کارمزد رایگان : امکان استفاده از پلن های کارمزد رایگان به مدت یک سال
تسویه حساب : امکان استفاده از تسویه حساب خودکار مستقیم (تسویه هر روز صبح ساعت 5 الی 7 صبح توسط خد شاپرک انجام می شود.)
درگاه به نام : امکان قرار دادن نام و لوگو مجموعه شما در صفحه ی شاپرک به جای نام و لوگوی پی پینگ

استفاده از فاکتور آنلاین پی پینگ

با استفاده از افزونه ووکامرس vip پی پینگ می توانید به صورت اتوماتیک از فاکتور آنلاین پی پینگ استفاده کنید. استفاده از فاکتور پی پینگ این امکان را برای شما فراهم می کند که موجودی محصولات شما به صورت اتوماتیک تغییر کند.

مدیریت موجودی ها

افرونه ووکامرس vip پی پینگ این امکان را به شما می دهد که موجودی انبار خود را چک و مدیریت کنید. به این صورت که در هنگام خرید
محصولات به صورت اتوماتیک موجودی از اکانت پی پینگ شما کسر می شود و به شما اطلاع داده می شود.
استفاده از این ویژگی باعث می شود که موجودی شما از هر طریقی که پرداخت می شود (وبسایت، فاکتور آنلاین و آیتم مالی ) مدیریت شود و به صورت اتوماتیک کسر شود.

صرفه جریی در زمان فقط با یک بار آپدیت

با استفاده از این افزونه محصولات جدید به صورت اتوماتیک به لیست اضافه می شوند و فقط یک بار کافیست لیست محصولات، شامل نام و قیمت را وارد کنید. همچنین تغییرات و ایجاد محصولات جدید را میتوانید به راحتی ایجاد و مشاهده کنید.

== Installation ==

1. پوشه 'woo-vip-payping-and-gateway' را در مسیر '/wp-content/plugins/' آپلود کنید
2. افزونه را از طریق منوی 'افزونه ها' در وردپرس فعال کنید
3. تنظیمات افزونه را می توانید از  قسمت تنظیمات ووکامرس / پیکربندی/ درگاه های پرداخت انجام دهید
